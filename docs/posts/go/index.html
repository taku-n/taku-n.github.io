Install

~/.local 以下にインストールしていくケースです

$ cd ~/.local
$ curl -OL https://go.dev/dl/gox.y.z.linux-amd64.tar.gz
$ tar zxf gox.y.z.linux-amd64.tar.gz
$ mkdir -p gopath/bin

.bashrc に以下を追記します

export PATH=~/.local/go/bin:~/.local/gopath/bin:$PATH
export GOPATH=~/.local/gopath
