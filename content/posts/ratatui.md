# Ratatui

## Installation

```
sudo aptitude update
sudo aptitude full-upgrade
sudo aptitude install pkg-config
sudo aptitude install libssl-dev

rustup self update
rustup update
cargo install cargo-generate
```

## Create a Project

```
cargo generate ratatui/templates simple
```
